    • JavaScript 1.2: Versión de los navegadres 4.0.

    • JavaScript 1.5: Versión actual que implementa Natescape 6.
    • JScript 5.5: Es la versión de Microsoft.
    • ECMAScript 5: Versión publicada en 2009 por ECMAScript, funciona en todos los navegadores del mercado.
    • ECMAScript 6: Lanzada en 2015, es la 6ta versión que contiene muchas novedades. Es compatible con todos los navgadores, excepto Internet Explorer

  3. Existen 3 maneras de declarar variables:
    • var: declara una variable iniciandola opcionalmente a un valor. Permite cambiar su valor, es decir sobreescribirla.
    • let: declara una variable local en un bloque de ámbito. También puede cambiar su valor.
    • const: declara variables de sólo lectura en un bloque. Su valor no puede ser cambiado mediante la asignación.

    4. Se debe tener en cuenta también que cuando se declara una variable fuera de una función se la denomina variable global, ya que estará disponible para cualquier otro código en el documento actual. Sin embargo, cuando la variable es declarada dentro de una función, ésta es denominada variable local, porque está disponible sólo dentro de la función donde fue creada.


  4. JavaScript tiene los siguientes tipos de datos:
    • string: secuencia de caracteres que representan un valor.
    • number: valor numérico.
    • boolean: valores true o false.
    • null: denota valor nulo.
    • undefined: valor sin definir.
    • symbol: tipo de dato, cuyos casos son únicos e inmutables.
    • object: objeto {} puede contener más variables en su interior.

  5. Expresiones regulares:

    Las expresiones regulares son patrones que se utilizan para hacer coincidir combinaciones de caracteres en cadenas. En JavaScript, las expresiones regulares también son objetos. Las expresiones regulares se utilizan con los métodos RegExp test() y exec() y con los métodos de String, match(), replace(), search() y split().


  6. Cómo declarar funciones en JavaScript:

    Una definición de función consta de la palabra clave function, seguida de: el nombre de la función, una lista de parámetros de la función, entre paréntesis y separados por comas, las declaraciones de JavaScript que definen la función, encerradas entre corchetes, { /* … */ }.

    function nom_funcion (parametros){ instrucciones; }


    Las funciones también se pueden crear mediante una expresión de función, esto es una sintaxis más amigable. Aquí se pueden usar las palabras reservadas var, let o const. Esto va proseguido por el nombre de la función, el signo igual (=), los parámetros o argumentos que van entre los paréntesis, una flecha formada por el signo igual y el signo menor (=>) y por último las llaves donde irán las instrucciones {...}.

    const nom_funcion = (parametros) => { instrucciones; }
