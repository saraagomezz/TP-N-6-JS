1. Existen diferentes versiones de Java Script
    • JavaScript 1: Nació con el Natescape 2.0.
    • JavaScript 1.1: Se diseñó con la llegada de los navegadores 3.0.
    • JavaScript 1.2: Versión de los navegadres 4.0.
    • JavaScript 1.3: Versión que implementan los navegadores 5.0.
    • JavaScript 1.5: Versión actual que implementa Natescape 6.
    • JScript 5.5: Es la versión de Microsoft.
    • ECMAScript 5: Versión publicada en 2009 por ECMAScript, funciona en todos los navegadores del mercado.
    • ECMAScript 6: Lanzada en 2015, es la 6ta versión que contiene muchas novedades. Es compatible con todos los navgadores, excepto Internet Explorer

  2. Cómo se enlaza JavaScript HTML:
    • Incluir la etiqueta "script" dentro de la etiqueta "head"
    • Incluir la etiqueta "script" dentro de la etiqueta "body"
    • Crear un enlace con el archivo de JavaScript dentro de la etiqueta "body"
