  3. Existen 3 maneras de declarar variables:
    • var: declara una variable iniciandola opcionalmente a un valor. Permite cambiar su valor, es decir sobreescribirla.
    • let: declara una variable local en un bloque de ámbito. También puede cambiar su valor.
    • const: declara variables de sólo lectura en un bloque. Su valor no puede ser cambiado mediante la asignación.

    Se debe tener en cuenta también que cuando se declara una variable fuera de una función se la denomina variable global, ya que estará disponible para cualquier otro código en el documento actual. Sin embargo, cuando la variable es declarada dentro de una función, ésta es denominada variable local, porque está disponible sólo dentro de la función donde fue creada.


    JavaScript tiene los siguientes tipos de datos:

    • string: secuencia de caracteres que representan un valor.
    • number: valor numérico.
    • boolean: valores true o false.
    • null: denota valor nulo.
    • undefined: valor sin definir.
    • symbol: tipo de dato, cuyos casos son únicos e inmutables.
    • object: objeto {} puede contener más variables en su interior.

    Expresiones regulares:

    Las expresiones regulares son patrones que se utilizan para hacer coincidir combinaciones de caracteres en cadenas. En JavaScript, las expresiones regulares también son objetos. Las expresiones regulares se utilizan con los métodos RegExp test() y exec() y con los métodos de String, match(), replace(), search() y split().


  4. Cómo declarar funciones en JavaScript:

    Una definición de función consta de la palabra clave function, seguida de: el nombre de la función, una lista de parámetros de la función, entre paréntesis y separados por comas, las declaraciones de JavaScript que definen la función, encerradas entre corchetes, { /* … */ }.

    function nom_funcion (parametros){ instrucciones; }


    Las funciones también se pueden crear mediante una expresión de función, esto es una sintaxis más amigable. Aquí se pueden usar las palabras reservadas var, let o const. Esto va proseguido por el nombre de la función, el signo igual (=), los parámetros o argumentos que van entre los paréntesis, una flecha formada por el signo igual y el signo menor (=>) y por último las llaves donde irán las instrucciones {...}.

    const nom_funcion = (parametros) => { instrucciones; }


    Esto incluso puede ser más resumido si nuestra función tiene un solo argumento, podemos dejar de usar los paréntesis y las llaves, simplemente después de nuestra flecha (=>) colocamos el argumento.

    const nom_function = (parametros) => instrucción;


  5. Existen dos tipos de operadores:

    Operadores de asignación:

    • adición
    • sustracción
    • multiplicación
    • división
    • resta

    Operadores de comparación:

    • Igualdad: devuele true si ambos operandos son iguales (==)
    • Desigualdad: devuelve true si ambos operandos no son iguales (!=)
    • Estrictamente iguales: devuelve true si ambos operandos son iguales y tienen el mismo tipo (===)
    • Estrictamente desiguales: devuelve true si los operandos no son iguales y/o no son del mismo tipo (!==)
    • Mayor que: devuelve true si el de la izquierda es mayor (>)
    • Mayor o igual que: devuelve true si el de la izquierda es mayor o igual (>=)
    • Menor que: devuelve true si el de la izquierda es menor (<)
    • Menor o igual que: devuelve true si el de la izquierda es menor o igual (<=)

    Además de los operadores aritméticosestándar, JavaScript brinda operadores aritméticos, los cuales son: resto (%), incremento (++) y decremento (--)


    También se puede mencionar el operador de concatenación, el cual une dos valores de string, devolviendo otro string correspondiente a la unión.

    Y el operador ternario, que necesita tres operandos: condición ? valor1 : valor2. Si la condición es true tomará el valor1, sino el valor2


    Convertir tipos de datos:

    Podemos usar parseInt() para parsear una cadena (string) e intentar obtener un valor numérico.

    También usamos parseFloat, que interpreta al número siempre como decimal.

  6. Estructuras de control de JavaScript:
    • if: Con él podemos indicar en el programa que se tome un camino sólo si se cumple la condición que establezcamos.
    • if/else: La sentencia if especifica un bloque de código que se ejecuta si una condición es cierta y da la oportunidad de ejecutar otro bloque de código distinto si resulta que no es cierta.
    • switch: Se utiliza para hacer diferentes acciones basadas en diferentes condiciones. Consiste en darle un valor y comprobar si ese valor coincide con algún caso. En el caso que coincida, ejecutamos un bloque de código específico.
    • for: Nos permite ejecutar el código un número determinado de veces, es decir un bucle que se repite hasta que la condición especificada sea false.
    • while: ejecuta sus sentencias mientras la condición sea verdadera.
